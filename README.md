## ✨ Fitur

- 🎯 **Set & Forget** - Input package name SEKALI, ingat selamanya!
- 🌍 **Universal** - Bisa buat app APAPUN (ujian, game, browser, media player)
- ⚡ **Pre-emptive Strike** - Lock volume SEBELUM app sempet bunyi
- 🎵 **Audio Focus Hijack** - Rebut hak suara dari target app
- 📋 **Live Log** - Pantau status realtime
- 🧹 **Auto Clear Log** - Fresh start setiap buka app

## 🚀 Cara Pake

1. Install APK
2. Input package name target (contoh: `net.sid.alguik_v1`)
3. Tekan **SAVE TARGET**
4. Tekan toggle ke **ON**
5. Minimize app (jangan close!)
6. Buka app target → **DIEM TOTAL!**

## 🔍 Cara Cari Package Name

1. Install **Package Name Viewer** dari Play Store
2. Cari app target
3. Copy package name-nya

## ⚠️ Catatan Penting

- Jangan close app selama mode ON (cukup minimize)
- Matikan toggle setelah selesai biar suara normal lagi
- App ini **TIDAK butuh root**!
